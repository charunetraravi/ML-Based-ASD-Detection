# ML-Based ASD Detection System

## 📌 Project Description
The ML-Based Autism Spectrum Disorder (ASD) Detection System predicts the likelihood of ASD using Machine Learning techniques.  
The system analyzes behavioral and screening data to assist early autism detection.

## 🛠 Technologies Used

- Jupyter Notebook
- Machine Learning
- Pandas
