# 🧠 ML-Based Autism Spectrum Disorder (ASD) Detection System

## 📌 Project Overview

The **ML-Based Autism Spectrum Disorder (ASD) Detection System** is a Machine Learning-based web application designed to predict the likelihood of Autism Spectrum Disorder using behavioral screening and questionnaire data.

This system assists in early ASD identification by analyzing user responses through trained machine learning models and providing instant prediction results through an interactive web interface.
